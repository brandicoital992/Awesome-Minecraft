A curated list of software, tools, and resources used in the Minecraft server ecosystem.  
This includes **server software, proxies, plugins, mod platforms, server panels, monitoring tools, developer tools, and Minecraft server lists** used to build and operate Minecraft servers and networks.

The goal of this repository is to provide a **central reference for Minecraft server owners, developers, and hosting providers**.

Contributions are welcome!

## Proxy

### Modern Minecraft Proxy Servers

- **[Velocity](https://github.com/PaperMC/Velocity)** - A modern, high-performance Minecraft proxy designed for scalability, flexibility, and security.

- **[BungeeCord](https://github.com/SpigotMC/BungeeCord)** - The original Minecraft proxy software used to connect multiple servers into a network.

- **[Waterfall](https://github.com/PaperMC/Waterfall)** - A high-performance fork of BungeeCord maintained by the PaperMC team.

- **[Travertine](https://github.com/PaperMC/Travertine)** - A fork of Waterfall that adds support for older Minecraft protocol versions.

- **[FlameCord](https://github.com/2lstudios-mc/FlameCord)** - A BungeeCord fork focused on performance, security, and exploit protection.

- **[HexaCord](https://github.com/HexaCord/HexaCord)** - A BungeeCord fork that improves security and performance for large networks.

- **[Aegis](https://github.com/Metrofico/BungeeAegisHD)** - A BungeeCord fork designed for DDoS protection and bot filtering.

- **[NanoLimbo](https://github.com/Nan1t/NanoLimbo)** - A lightweight limbo server used with proxies to handle bot attacks and queue systems.

### Legacy / Experimental Proxies

- **[BungeeGuard](https://github.com/lucko/BungeeGuard)** - A proxy security plugin used to protect backend servers from unauthorized connections.

- **[RedisBungee](https://github.com/RedisBungee/RedisBungee)** - A system allowing multiple BungeeCord proxies to communicate using Redis.

- **[Waterdog](https://github.com/WaterdogPE/Waterdog)** - A proxy software designed to connect Bedrock servers together.

- **[WaterdogPE](https://github.com/WaterdogPE/WaterdogPE)** - A proxy allowing Bedrock Edition servers to connect in a network.

- **[Geyser](https://github.com/GeyserMC/Geyser)** - A proxy bridge that allows Bedrock Edition players to join Java Edition servers.

- **[Floodgate](https://github.com/GeyserMC/Floodgate)** - A proxy extension allowing Bedrock players to join Java servers without needing a Java account.
