## 🚀 How to use it

1. Open the repository page.
2. Read the list of tools or topics.
3. Click the tool name you need.
4. Open the linked page for that tool.
5. Download the file from the tool’s own page if needed.
6. Follow the setup steps on that tool’s page.

If you are new to server setup, start with:

- A server jar like PaperMC or Spigot
- A permissions tool like LuckPerms
- A chat and command tool like EssentialsX
- A world tool like WorldEdit or WorldGuard

That gives you a simple base for a private or public server.

## 🧰 Common use cases

### 🏗️ Build a server
Use server software links to choose the base for your Minecraft world.

### 🔀 Connect servers
Use proxy tools like BungeeCord or Velocity if you want more than one server in one network.

### 🧩 Add features
Use plugins like Citizens, EssentialsX, and McMMO to add game features.

### 🗺️ Manage worlds
Use WorldEdit, WorldGuard, and Chunky to edit land, protect areas, and prepare maps.

### 👥 Control access
Use LuckPerms to set player roles and permissions.

### 🔄 Support versions
Use ViaVersion to help players join from different Minecraft versions.

## 🛠️ Basic setup flow for non-technical users

If you want the simplest path, use this order:

1. Choose a server base.
2. Add a permissions plugin.
3. Add player tools.
4. Add world tools.
5. Test the server with one account.
6. Add more tools only after the server works.

This keeps setup simple and helps you avoid too many changes at once.

## 🔍 How to pick the right tool

Use this quick guide:

- Need a stable server base: PaperMC
- Need plugin support: Spigot or Bukkit
- Need multiple servers linked together: Velocity or BungeeCord
- Need player permissions: LuckPerms
- Need commands and kits: EssentialsX
- Need region protection: WorldGuard
- Need building tools: WorldEdit
- Need NPCs: Citizens
- Need skills and leveling: McMMO
- Need chunk preloading: Chunky
- Need version support: ViaVersion
